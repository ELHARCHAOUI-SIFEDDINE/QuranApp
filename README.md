## Installation

1. Clone the repository

```bash
git clone https://github.com/ELHARCHAOUI-SIFEDDINE/QuranApp.git
cd quran-app
```

2. Install dependencies

```bash
npm install
# or
yarn install
```

3. Start the development server

```bash
npm start
# or
yarn start
```
